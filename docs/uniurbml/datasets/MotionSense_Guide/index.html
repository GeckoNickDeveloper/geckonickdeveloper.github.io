MotionSense

MotionSense Datasets Guide

MotionSense is a dataset for Human Activity Recognition (HAR), sampled at 50Hz containing accelerometer and gyroscope data. The following files contains data obtained by 24 subjects.

Each record is composed by:

Accelerometer Gyroscope Label Subject Weight Height Age Gender Trial
x - y - z x - y - z label-id subject-id weight height age gender trial

All data is represented as float64

The dataset file

WISDM.npy

Label Activity
0 Stairs Dwon
1 Stairs Up
2 Walking
3 Jogging
4 Standing
5 Sitting

Import

In order to import the dataset, you just need to type:

import numpy as np
dataset = np.load('MotionSense.npy')
